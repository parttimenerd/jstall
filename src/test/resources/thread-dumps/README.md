# Thread Dump Test Resources

This directory contains sample thread dumps for testing.

## Generated Dumps

These dumps are generated by running `ThreadDumpGenerator.java`:

- `deadlock.txt` - Thread dump with a classic deadlock scenario
- `busy-work-*.txt` - Multiple dumps showing busy threads doing CPU work
- `normal-*.txt` - Normal application dumps

## Generating Dumps

To regenerate the test dumps:

```bash
mvn test -Dtest=ThreadDumpGenerator#generateAllDumps
```

Or run individual generators:

```bash
mvn test -Dtest=ThreadDumpGenerator#generateDeadlockDump
mvn test -Dtest=ThreadDumpGenerator#generateBusyWorkDumps
mvn test -Dtest=ThreadDumpGenerator#generateNormalDumps
```

## Using in Tests

Load dumps in tests using the ThreadDumpProvider:

```java
Path dumpFile = Paths.get("src/test/resources/thread-dumps/deadlock.txt");
List<ThreadDump> dumps = provider.loadFromFiles(List.of(dumpFile));
```
